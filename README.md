# Flask test project

## Getting started

--------

1. Create .env and fill this
***
    HOST=
    PORT=
    FLASK_ENV=
    DEBUG=
    SECRET_KEY=
***
2. Create venv, activate it and upgrade pip:
***
    python3 -m venv venv
    . ./venv/bin/activate
    pip install -U pip
***
3. Install dependencies:
***
    pip install -r requirements.txt
***
4. Start app:
***
    flask run
***
5. Done
